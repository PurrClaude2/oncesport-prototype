# Прототип — видеоредактор для разбора тренировок (OnceSport)

Тип: **3 — рабочий аналог** (все ключевые флоу реально работают, без мок-данных).
Стек: Vite + React, чистый CSS (тёмный профи-UI, без shadcn). Видео обрабатывается **локально в браузере** (objectURL), бэкенда нет.
Референс: OnceSport (тёмный профи-концепт монтажки) + Purrweb-кит.

## Что реально работает
- **Проекты** — библиотека проектов (команда/клиент), сохраняется в localStorage.
- **Плеер + таймлайн** — загрузка видео, пауза, покадрово (← →), скорость 0.25/0.5/1/2x, перемотка кликом, вкл/выкл оригинального звука.
- **Инфографика (canvas поверх видео)** — стрелка, кольцо, линия, свободное перо; цвет, толщина; undo/clear; шорткаты 1-4.
- **Нарезка + теги** — точки In/Out (I/O), создание фрагмента с тегом (атака/оборона/вброс/другое), авто-раскладка по папкам, переход к фрагменту, бэнды на таймлайне.
- **Комментарии** — текстовые заметки к моменту, клик → переход к времени.
- **Брендирование** — стилизованная плашка снизу (название + тренер), впечатывается в экспорт.
- **Экспорт видео** — реальная запись (canvas.captureStream + MediaRecorder) итогового фрагмента/всего видео с наложенной графикой и брендингом → скачивание .webm. Экспорт всего разбора или отдельного фрагмента.

## Запуск
```
npm install
npm run dev      # дев-сервер
npm run build    # прод-сборка в dist/
npm run preview  # предпросмотр прод-сборки
```

## Деплой
Веб → Vercel. Нужен доступ Vercel (CLI/аккаунт). Варианты:
- `npx vercel` в этой папке (после `vercel login`);
- или закинуть папку `dist/` на любой статик-хостинг.

## Заметки
- Экспорт и `captureStream` стабильнее всего в Chrome.
- Шорткаты: V выбор · 1 стрелка · 2 кольцо · 3 линия · 4 перо · I/O начало-конец · ← → кадр · пробел плей · Ctrl+Z отмена.
- Трекинг/проекция камеры (премиум-графика из сметы) в прототип не входят — это тяжёлый CV-блок отдельной разработки.
